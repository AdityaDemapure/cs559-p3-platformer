# Celeste-ish 3D Dash Platformer (THREE.js)

- Prototype mode: primitive geometry only, no textures.
- Full mode: loads a GLTF model and loads/applies textures.
- Controls: WASD/Arrows, Space jump, Shift (or K) dash, drag to rotate camera.

## Local run
From this folder:
- `python -m http.server 8000`
Open: http://localhost:8000

## GitHub Pages
Repo Settings → Pages → Deploy from a branch → main → /(root).
